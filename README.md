# pinata-ui-react-native

An example of using https://github.com/pinata-ai/pinata-ui in React-Native application.

## Installation

```sh
npm install pinata-ui-react-native
```

## Usage


```js
import { PinataRewardsFlow } from 'pinata-ui-react-native';

// ...

<PinataRewardsFlow />
```
